Colour palette tester

Paste your colour palette:
At present the paste function only supports HEX, RGB and HSL colour formats
Each colour must be on it's own line
The contrast checker does not support transparency, as such, RGBA and HSLA colour formats as well as HEX codes longer than 6 characters (or 7 if including the '#') will not work
RGB and HSL colours must be in the format rgb(r, g, b) or hsl(h, s, l)
Make sure there are no extra characters at the end of each line (e.g. commas, semi-colons, etc.)
